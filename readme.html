README - Bimantoro Personal Website

1. Struktur HTML

Halaman utama (`index.html`) terdiri dari:

  • <header> → Judul halaman.
  • <div class="container"> → Semua konten utama seperti foto profil, hobi, link, tombol Gaskeun, dan tombol musik.
  • <img class="foto-profil"> → Foto profil dengan animasi goyang.
  • <ul> → Daftar hobi dengan animasi looping (fade-in + glow + bounce).
  • <audio id="musik"> → Musik background autoplay & loop.
  • <button id="klikBtn"> → Tombol Gaskeun 😎 yang trigger efek warna, teks, overlay, dan redirect YouTube.
  • <button id="musikBtn"> → Tombol play/pause musik.
  • <div id="overlay"> → Overlay hitam untuk efek fade-out saat Gaskeun ditekan.

2. Styling CSS (`style.css`)

CSS bertugas untuk tampilan & animasi:

  • body → Background gambar fixed, font, warna teks.
  • .container → Box transparan, border-radius, margin & padding.
  • .foto-profil → Animasi goyang (keyframes goyang).
  • ul li → Animasi hobi: fadeInUpLoop, glow, bounce, delay per item.
  • button → Styling tombol Gaskeun & musik (hover effect).
  • #overlay → Overlay hitam fade-in saat tombol Gaskeun ditekan.

3. JavaScript (`script.js`)

JS mengontrol interaktivitas:

  • Musik Play/Pause: toggle play dan update teks tombol.
  • Tombol Gaskeun 😎:
    • Ganti background ke warna random.
    • Tambah teks "Gaskeun! 🚀" ke halaman.
    • Aktifkan overlay hitam dan mainkan suara whoosh.
    • Redirect ke playlist YouTube setelah 1.5 detik.
  • Animasi Hobi: Saat load, setiap <li> hobi diatur opacity dan animasi looping via CSS.

4. Struktur File

/project-root
│
├─ index.html       # Halaman utama
├─ style.css        # Styling dan animasi
├─ script.js        # Logika tombol & animasi
└─ readme.html      # Halaman README ini

5. Cara Pakai

  1. Clone atau download project.
  2. Pastikan semua file berada di folder yang sama.
  3. Buka index.html di browser.
  4. Nikmati animasi, musik, tombol Gaskeun, dan hobi yang beranimasi.
