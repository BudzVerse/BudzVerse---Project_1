README - Bimantoro Personal Website

Fitur

  • Profil Diri: Foto profil dengan animasi goyang
  • Hobi: Fade-in + Glow warna-warni + Bounce (looping)
  • Musik Autoplay: Bisa play/pause
  • Tombol Gaskeun 😎: Background random + teks "Gaskeun! 🚀" + Fade-out + Suara whoosh + Redirect YouTube
  • Link WhatsApp: Langsung chat

Struktur File

/project-root
│
├─ index.html
├─ style.css
├─ script.js
└─ readme.html

Cara Pakai

  1. Clone atau download project
  2. Pastikan file berada di folder sama
  3. Buka index.html di browser
  4. Nikmati animasi, musik, dan tombol Gaskeun 😎!

Kustomisasi

  • Ganti musik di <audio id="musik">
  • Ganti foto profil di <img class="foto-profil">
  • Tambah hobi di <ul>
  • Ubah tombol Gaskeun redirect di script.js
